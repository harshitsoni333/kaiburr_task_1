Jar file to run the project is located in:
usermanagement\target\

Methods used in the project:

GET localhost:8080/servers			// returns list of servers
GET localhost:8080/servers/<id or name>		// returns a server by name or id

DELETE localhost:8080/servers/<id>		// deletes server by id
PUT localhost:8080/servers			// accepts a new server in json format and saves in database
